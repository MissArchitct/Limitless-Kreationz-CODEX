This `vault/` directory is **documentation-only** in this GitHub repo.

- lives **locally on-device** (iOS app),
- is encrypted with **AES-256**,
- is gated by **biometric-only access** (Touch ID / Face ID),
- never syncs to cloud storage by default.

Use this folder to store design notes, not real secrets.
